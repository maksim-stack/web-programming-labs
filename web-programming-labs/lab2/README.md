# Лабораторна робота №2 — Типізація проєкту

**Варіант:** 3

## Структура проєкту

- `src/task-1.ts` — Базові типи, інтерфейси та type aliases
- `src/task-2.ts` — Generics та Utility Types
- `src/task-3.ts` — Класи та модифікатори доступу
- `src/task-4.ts` — Наслідування та поліморфізм
- `src/task-5.ts` — Type Guards та звуження типів

## Запуск

1. Встановіть залежності: `npm install`
2. Скомпілюйте TypeScript: `npx tsc`
3. Запустіть завдання: `node dist/task-N.js`
